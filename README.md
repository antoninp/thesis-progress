# Thesis progress

Live progress dashboard for my Master's thesis — auto-updated.

**View it here:** https://antoninp.github.io/thesis-progress/

- **Progress** (`index.html`) — word count, schedule, pace, and chapters.
- **Full history** (`history.html`) — day-by-day writing and estimated hours.

This repository contains only the generated dashboard pages (aggregate numbers,
dates, and chapter titles — no thesis text). It is refreshed automatically.
